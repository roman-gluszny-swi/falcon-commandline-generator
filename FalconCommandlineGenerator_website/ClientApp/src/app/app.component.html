Guide

  • Set product/component parameters
  • Set additional parameters/select catalog stage

Working features

  • Adding/removing/updating products and components (names and build numbers dynamically loaded from PC API
  • Update only parameter for each item
  • Additional commandline parameters
  • Edit commandline
  • Copying to clipboard
